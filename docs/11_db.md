# Database (Postgres)

## Minimal schema
The schema lives in `schemas/postgres.sql`.

## Quick setup
1. Create a Postgres database.
2. Run the SQL schema.
3. Set `DATABASE_URL` in `.env`.

## Example URL
`postgres://ember:ember@localhost:5432/ember`

## Project auth
Each project has an `api_key`.
The key is created on first ingestion (if the project does not exist).

## Admin
Admin operations (create/rotate keys) use `EMBER_SECRET`.
